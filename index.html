Features

코딩부터 실물 제어까지, 하나의 플랫폼

초등학생부터 중학생까지, 단계별 학습 경로로 자동화의 핵심을 배울 수 있습니다.

🧩

PLC 카드 코딩

드래그앤드롭으로 53종 카드를 조합하여 자동화 흐름을 설계합니다. 4단계 학습 단계로 차근차근 학습할 수 있습니다.

🤖

AI 어시스턴트

자연어로 원하는 동작을 설명하면 블록을 자동 생성합니다. AI 디버깅으로 오류도 쉽게 해결하세요.

📷

AI 비전 스튜디오

5종 ML/DL 알고리즘으로 이미지 분류 모델을 직접 학습하고, 데이터 분석 차트와 PCA 시각화로 AI를 이해합니다.

📝

IEC 61131-3 ST

산업 표준 PLC 언어인 구조화 텍스트(ST)를 블록과 함께 학습합니다. CODESYS, TIA Portal 포맷으로 내보내기 가능합니다.

System Requirements

시스템 요구사항

시뮬레이션 모드로 하드웨어 없이도 모든 기능을 체험할 수 있습니다.

💻 소프트웨어

  • macOS 12 Monterey 이상 (Apple Silicon / Intel)
  • Windows 10 / 11 Coming Soon
  • 디스크 공간 500MB 이상
  • RAM 4GB 이상 (8GB 권장)

🔧 하드웨어 (선택)

  • UFactory Lite 6 로봇팔
  • ESP32 + TB6600 스텝모터 컨베이어
  • ESP32-CAM (MJPEG 스트림)
  • E18-D80NK 적외선 센서

📡 네트워크

  • 로봇: LAN 직결 (192.168.1.x)
  • 카메라: Wi-Fi 연결
  • 컨베이어: USB Serial
  • 인터넷 연결 불필요 (오프라인 사용 가능)
