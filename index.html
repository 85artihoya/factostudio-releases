v1.7.0 — 전체 릴리즈 목록

Features

하나의 플랫폼에서
AI부터 로봇 제어까지

데이터 수집, AI 학습, 판단, 로봇 행동까지 — 피지컬 AI의 전체 파이프라인을 단일 환경에서 경험합니다.

Phase 1

팩토블록 코딩

PLC 자동화 개념을 블록으로 추상화한 신개념 코딩. 53종 블록을 조합하여 IF/LOOP/WHILE 등 프로그래밍 핵심 개념을 실물 장비와 함께 학습합니다.

Phase 2

AI 비전 스튜디오

카메라로 데이터 수집부터 KNN/DNN/SVM 등 5종 알고리즘 학습, PCA 시각화까지. 브라우저에서 직접 AI 모델을 만듭니다.

v2.0 Coming

노드 워크플로우

AI 노드 + 하드웨어 노드를 자유롭게 연결하는 비주얼 캔버스. KitProfile 시스템으로 다양한 하드웨어 키트를 지원합니다.

IEC 61131-3

ST 코드 에디터

산업 표준 구조화 텍스트(ST) 언어를 블록과 함께 학습합니다. CODESYS, TIA Portal SCL 포맷으로 내보내 실제 PLC에 적용 가능합니다.

Data

데이터 분석

수집된 센서 데이터와 이미지를 차트(히스토그램/히트맵/산점도)로 시각화하고, 데이터의 균형과 분포를 분석합니다.

Safety

교육용 안전 시스템

E-STOP 긴급 정지, 속도 제한, 모터 과열 보호, 실행 전 안전 검증까지. 교육 환경에 최적화된 다중 안전 장치를 제공합니다.

Physical AI Pipeline

데이터에서 행동까지, 하나의 흐름

센서 데이터 수집 → AI 학습 → 실시간 판단 → 로봇 행동. 피지컬 AI의 전체 파이프라인을 시각적으로 설계합니다.

Hardware Kits

다양한 하드웨어 키트 지원

KitProfile 시스템으로 하드웨어를 자유롭게 확장합니다. 로봇팔 키트를 시작으로, 다양한 산업 시나리오를 지원할 예정입니다.

🦾

로봇팔 분류 키트

6축 협동 로봇팔 + 컨베이어 벨트 + AI 카메라 + 센서. AI 비전으로 물체를 판별하고, 로봇팔이 위치별로 분류하는 스마트 팩토리 라인입니다.

자율주행 로봇(AGV)이 경로를 탐색하며 물류를 운반합니다. 경로 최적화와 장애물 회피를 AI로 학습합니다.

계획 중 자율주행
Learning Path

초등학생부터 산업 현장까지

같은 플랫폼에서 단계를 올리며 성장합니다. 팩토블록에서 시작해 AI + 하드웨어를 통합하는 전문가로.

1

팩토블록

초등 고학년

PLC 자동화를 블록으로 추상화. 드래그앤드롭으로 순차/반복/조건 등 프로그래밍 기초를 학습합니다.

2

AI 비전

중학생

카메라로 데이터를 수집하고 AI 모델을 직접 학습시켜 분류 자동화를 구현합니다.

3

ST 코드

고등학생

산업 표준 IEC 61131-3 ST 언어로 텍스트 코딩에 입문합니다. 실제 PLC와 호환됩니다.

4

노드 워크플로우

대학 / 산업

AI + 하드웨어를 노드로 연결하는 비주얼 프로그래밍. 실제 산업 자동화 시나리오를 설계합니다.

Requirements

시스템 요구사항

시뮬레이션 모드로 하드웨어 없이도 AI 학습부터 블록 코딩까지 모든 기능을 체험할 수 있습니다.

💻 소프트웨어

  • macOS 12 Monterey 이상 (Apple Silicon / Intel)
  • Windows 10 / 11 (64-bit)
  • 디스크 공간 500MB 이상
  • RAM 4GB 이상 (8GB 권장)

🔧 하드웨어 키트 (선택)

  • 6축 협동 로봇팔
  • 컨베이어 벨트 (스텝모터)
  • AI 카메라 (실시간 스트리밍)
  • 적외선 감지 센서

🌐 연결 방식

  • 로봇팔: LAN 케이블 직결
  • AI 카메라: Wi-Fi 무선 연결
  • 컨베이어: USB 케이블 연결
  • 인터넷 연결 불필요 (오프라인 사용 가능)
